## 2. LinkedIn Algorithm 2026 - What Works

### Critical Changes
- **Quality over quantity**: "Expertreneurs" with proven results win
- **Dwell time is king**: 30+ seconds engagement = distribution boost
- **Proof, not polish**: Screenshots/case studies > generic advice
- **Topic consistency**: Algorithm rewards clear expertise area

### What Gets Engagement
- Text posts: Build authority
- Carousels: Break complex ideas into steps (high saves/shares)
- Short videos: Highest engagement for demos
- First 210 characters: Critical (before "See More")

## 6. Best Practices for Technical Demos

### What Works
1. **Lead with result**: Show demo/output first
2. **Concrete metrics**: "Processes X tasks in Y seconds"
3. **Visual proof**: Screenshots, code, demo videos
4. **Technical depth**: Don't dumb it down
5. **Open source**: GitHub repo, documentation, live demo
6. **Problem-solution**: What problem does it solve for Macrohard/xAI?
