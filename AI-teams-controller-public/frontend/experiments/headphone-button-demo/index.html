How to Test

  1. Connect your Bluetooth headphones to this device
  2. Click "Initialize Media Session" button above
  3. Press the play/pause button on your headphones
  4. Watch the mic indicator toggle on/off
  5. Check the event log for button press events

Note: This uses the Media Session API with a silent audio track. Headphone buttons trigger play/pause which toggles voice input state.
