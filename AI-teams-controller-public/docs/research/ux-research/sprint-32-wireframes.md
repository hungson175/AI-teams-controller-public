## Accessibility Features

### Keyboard Navigation
```
Tab sequence:
1. Mode toggle (Design/Monitor)
2. Team selector (if Monitor mode)
3. Template selector (if Design mode)
4. Save button
5. Load button
6. Canvas nodes (focusable)
7. Edges (focusable)

Shortcuts:
- M: Toggle mode
- T: Focus team selector (Monitor mode)
- Esc: Exit modals/dropdowns
```

### Screen Reader Announcements
```
On mode switch to Monitor:
  "Switched to Monitor mode. Select a team to view live status."

On team selection:
  "Monitoring ai_controller_full_team. 4 roles active, 2 idle."

On status change:
  "PM role is now active."
  "Backend Engineer role is now idle."
```

## Animation Specifications

### Status Dot Pulse (Active)
```css
@keyframes pulse-active {
  0% {
    opacity: 1;
    transform: scale(1);
    box-shadow: 0 0 0 0 rgba(34, 197, 94, 0.7);
  }
  50% {
    opacity: 0.8;
    transform: scale(1.05);
    box-shadow: 0 0 0 4px rgba(34, 197, 94, 0);
  }
  100% {
    opacity: 1;
    transform: scale(1);
    box-shadow: 0 0 0 0 rgba(34, 197, 94, 0);
  }
}

Duration: 2s
Easing: ease-in-out
Iteration: infinite
```

### Mode Switch Transition
```css
.mode-content {
  transition: opacity 200ms ease-in-out;
}

.mode-content[data-state="inactive"] {
  opacity: 0;
  pointer-events: none;
}

.mode-content[data-state="active"] {
  opacity: 1;
  pointer-events: auto;
}
```

### Team Selector Slide
```css
.team-selector {
```
